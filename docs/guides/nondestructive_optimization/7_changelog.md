# Список изменений

## v0.2.0

- Гайд добавлен на сайт
- Улучшена навигация по разделам
- Улучшено визуальное отображение названий компонентов и выделены предупреждения и примечания
- В разделе с оптимизацией мешей добавлены инструкции как сделать `Basic Mesh` из `Skinned Mesh`, и как восстановить удаленные меши при необходимости
- Для слоев аниматора в прочих оптимизациях добавлена ссылку на англоязычный сайт по переносу тогглов в один слой аниматора в Blend Tree, а также приведены примеры вещей, которые так перенести не получится
- Добавлены лимиты для Android-версии аватара
- Добавлены ссылки на другие части гайда в подходящих местах


## v0.1.1

- Исправлены ссылки в PDF-версии
