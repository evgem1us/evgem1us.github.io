# Коллайдеры (PhysBone Colliders, PhysBone Collision Check Count)

# Описание

Коллайдеры не дают PhysBones проходить сквозь части аватара, к котором привязаны.  Коллайдером является компонент `VRC Phys Bone Collider`. 

## Уменьшение PhysBone Colliders

`PhysBone Colliders` уменьшаются вручную удалением лишних коллайдеров. 

В некоторых случаях `AAO Trace and Optimize` объединит коллайдеры с одинаковыми размерами и прочими настройками, но это может сработать далеко не всегда. Если такие остаются (видно при осмотре аватара в Play Mode или при [ручном запекании аватара](../../prepare#nondestructive)), можно вручную задать один из коллайдеров в настройках PhysBone, а другой удалить (для PhysBones, объединяемых через `AAO Merge PhysBone`, можно выбрать действие `Override` и добавить нужные коллайдеры в список вручную).

## Уменьшение PhysBone Collision Check Count

Оптимизировать данную характеристику можно только выполнением всех предыдущих действий с PhysBones (все описанное в [разделе PhysBones](.)), для которых заданы коллайдеры.
