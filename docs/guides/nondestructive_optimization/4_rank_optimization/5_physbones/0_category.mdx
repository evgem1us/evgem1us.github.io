# PhysBones

PhysBones нагружают процессор и управляют физикой волос, кошкочастей тела, физику одежды, и т.п.

Для оптимизации нужно уменьшать:
- `PhysBone Components` - Количество PhysBone-компонентов на аватаре 
- `PhysBone Transforms` - Общее количество костей, которыми управляют все PhysBones на аватаре
- `PhysBone Colliders` - Количество коллайдеров в PhysBone-компоненте, с которыми взаимодействуют кости 
- `PhysBone Collision Check Count` - Количество костей в PhysBone-компоненте, которые взаимодействуют с коллайдерами

Выделить все PhysBones на аватаре можно через VRChat SDK, нажав на `Select` напротив `Phys Bone Components` или `Phys Bone Transform Count`.

Увидеть все управляемые PhysBones кости и коллайдеры можно, включив Gizmos (сфера в правом верхнем углу на сцене):
- Белым цветом будут обозначены границы костей, управляемые PhysBone
- Светло-голубым (или какой это) будут показаны границы поворота каждой кости, управляемой PhysBone
- Зеленым обозначены границы коллайдеров:
  - Зеленый квадрат - это Plane-коллайдер, т.е. плоскость, и ее размер бесконечен. Коллайдиться будет все, что соприкасается с этой плоскостью, а еще у нее есть направление (обозначается стрелочкой)
