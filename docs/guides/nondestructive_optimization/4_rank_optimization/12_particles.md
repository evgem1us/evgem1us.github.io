# Частицы (Particle Systems, Total Max Particles, Mesh Particle Triangles)

С частицами я особо не работал, поэтому ограничусь небольшой выдержкой из документации VRChat-а и парой вещей (могут быть неточными):

1. Каждый `Particle System` занимает один материал, плюс еще один если используются `Trails`
2. Чтобы ограничить кол-во частиц (`Total Max Particles`), поменяйте параметр `Max Particles` в `Particle System`
3. Если в качестве частицы используется меш (`Particle System` - `Renderer` - `Render Mode`=`Mesh`) и нужно уменьшить общее количество треугольников на нем (`Mesh Particle Triangles`, считается как макс. "кол-во частиц х треугольники меша") - уменьшите максимальное количество частиц, либо уменьшите количество треугольников для меша в 3D-редакторе типа Blender-а. Сомневаюсь что через Unity можно недеструктивно сжать меш в частицах
4. Если не укладываетесь в количество `Particle Systems`, удалите лишние (логично)
