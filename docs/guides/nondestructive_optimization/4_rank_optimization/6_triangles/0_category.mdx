# Triangles (треугольники, полигоны)

## Описание

Полигоны представляет собой плоскую геометрическую фигуру, состоящую из нескольких точек - т.н. вертексов - соединенных отрезками (ребрами).  
Любой полигон в 3D-модели минимально состоит из трех вертексов, которые образуют треугольник.  
Любой полигон можно разбить на несколько треугольников (например квадратный полигон состоит из двух треугольников).  

:::warning
Ранг производительности считает в аватаре именно количество треугольников.
:::

Cамая большая проблема современных моделей для аватаров - довести кол-во треугольников до 70 тысяч чтобы получить хотя бы `Poor` в ранге производительности. Хотя видеокарты сейчас могут без проблем кушать больше треугольников и не подавиться.

Используемые в гайде пакеты позволяют удалять полигоны с аватара недеструктивно прямо в Unity. Такое удаление полигонов не может (пока что) заменить полностью использование 3D-редактора, но позволяет оставить для редактора только самую сложную и долгую работу.

В разделах по удалению полигонов приведены разные компоненты, которые настраиваются немного по разному. Выбирайте любой понравившийся.
