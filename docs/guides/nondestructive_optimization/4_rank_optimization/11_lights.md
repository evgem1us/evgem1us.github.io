# Lights (Источники освещения)

Если на вашем аватаре должно быть много `Lights` - оптимизация на нем вам особо не нужна, он должен выглядеть максимально хорошо (грузите такую версию аватара отдельно от основной ради б-га), хотя сделать для него оптимизацию по гайду тоже не помешает.

А в целом можно только один источник света для Poor модели, но оставляйте его только если точно знаете, что делаете, и держите выключенным когда он не нужен. Красивая подсветка аватара того не стоит, лучше настроить шейдер на материале.
