# Краткая версия гайда

1. Добавьте на корень аватара компонент `AAO Trace and Optimize`, компонент объединит меши и удалит ненужное
2. `VRCFury` используйте по минимуму, его компоненты неизвестны для оптимизатора и изменяться не будут. Если нужно прикрепить одежду или сделать тоггл - `Modular Avatar` или ручками в аниматоры
3. Удалите ненужные меши на аватаре или поставьте им в инспекторе тег с `Undefined` на `Editor Only` и выключите.
4. Добавляйте только необходимый минимум тогглов и анимаций включения/выключения частей модели (волосы, одежда), включаемые/выключаемые меши объединены не будут.
5. Найдите все PhysBones на аватаре (VRChat SDK может подсветить их при нажатии на `Phys Bone Components` в `Review Any Alerts`) аватара добавьте `AAO Merge PhysBone` и добавьте в список PhysBones, находящиеся под одной костью.
   1. При необходимости уменьшите количество костей, управляемых PhysBone, с помощью добавления компонента `AAO Merge Bone` (влияет на параметры аватара PhysBone Transforms count и PhysBone Collider Transforms). Компонент объединит текущую кость с родительской. Ставьте компонент на кости через одну или два через один.
6. Удалите невидимые под одеждой полигоны с помощью компонентов `AAO Remove Mesh by ...`, `MA Shape Changer` в режиме `Delete` или `MA Mesh Cutter` + `MA Vertex Filter`.
7. При необходимости сожмите аватар по полигонам - нажмите на аватар ПКМ - `Meshia Mesh Simplifier` - `Meshia Cascading Avatar Mesh Simplifier` и поиграйтесь с настройками в одноименном объекте в аватаре
8. Зайдите в аниматор FX аватара, удалите ненужные слои с ненужными тогглами оригинальной одежды, волос, частей тела, контактов
9. Соберите аватар с помощью ПКМ на аватаре - `Modular avatar` - `Manual Bake Avatar`, посмотрите в VRChat SDK на результат (у собранного аватара будет `(Clone)` в конце) и на оставшиеся в аватаре компоненты на сцене. Оптимизируйте лишние на оригинальном аватаре.
10. Проверьте аватар через Gesture Emulator.
